**空间气象 Python 绘图实战手册：解决bug、复用轮子、沉淀经验的一站式工具箱**
## 📖 项目简介
这是一份为空间气象/极光数据可视化打造的 Python 绘图实战手册，帮你解决这些痛点：
- ✅ 空间气象数据绘图的常见bug（投影转换、配色失真、坐标偏移等）
- ✅ 开箱即用的可复用轮子（Matplotlib/ Cartopy 模板、配色方案、数据处理脚本）
- ✅ 踩坑经验与最佳实践（数据预处理、性能优化、配色规范）
- ✅ 可直接套用的配色方案（适配空间数据的专业色卡，解决可视化效果差的问题）
- ✨ 项目亮点
- 📦 **可直接复用**：每个案例都附带完整可运行代码，复制粘贴就能跑通
- 🐛 **踩坑实录**：记录了我在空间气象绘图中遇到的所有bug和解决方案
- 🎨 **专业配色**：整理了适配空间数据的配色方案，告别“辣眼睛”的可视化
- 📚 **持续更新**：不断补充新的案例、轮子和优化技巧
1 map-project 提供中国行政区坐标系和边框导入方案：我的优势有两个（1） 因为第三方库他们大多缺少台湾和香港 不能用 （2）中国官方的数据网提供的shp读取起来太麻烦数据点太多 我压缩成csv 并且提供示例代码告诉大家如何导入
